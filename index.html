关于投资日历

项目目标

聚合多个主流投资平台的事件数据，为投资者提供统一的事件日历视图，帮助把握投资机会。

数据源

  • 财联社 - 经济数据、事件公告
  • 韭研公社 - 投资事件分析
  • 同花顺 - 股票相关事件
  • 英为财情 - 国际经济数据
  • 东方财富 - A股市场数据

更新机制

通过GitHub Actions每日自动采集数据，检测变更并更新网站。所有数据处理过程完全透明。

技术栈

  • 数据采集：Python + Requests + BeautifulSoup
  • 前端：HTML5 + CSS3 + Vanilla JavaScript
  • 部署：GitHub Pages + GitHub Actions
